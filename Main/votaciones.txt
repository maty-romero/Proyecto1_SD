Desglosando las reglas:Lista de votos

-->Respuesta inválida:
Si la mayoría de los votos es False, la respuesta es inválida (0 puntos).
en caso de empate en votacion puntaje 0 y es invalida

-->Respuesta válida:
Si la mayoría de los votos es True, la respuesta es válida.
Si una respuesta es única (es decir, ningún otro jugador la dio), el jugador recibe 10 puntos.
Si una respuesta está repetida por otros jugadores, todos los jugadores con esa respuesta reciben 5 puntos.




1)Votación de los jugadores: Los jugadores deben poder votar sobre las respuestas de los demás. Cada cliente enviará un voto (1 o 0) al servidor.

2)Servidor calcula la validez de las respuestas:
Si la mayoría de los votos son True, la respuesta es válida.
Si hay empate, la respuesta es inválida.
Si la respuesta es válida, se asignan puntos según si es única o repetida.
3)Envío de resultados a todos los clientes: Después de 
calcular la validez de la respuesta, el servidor debe enviar 
la información a todos los clientes.

def calcular_puntajes(self):
    puntajes = {jugador: 0 for jugador in self.respuestasDict.keys()}

    for categoria, respuestas_jugadores in self.respuestasDict.items():
        # Paso 1: evaluar votos
        validas = {}
        for jugador, respuesta in respuestas_jugadores.items():
            votos = self.votosDict[categoria].get(jugador, [])
            total_true = sum(1 for _, v in votos if v)
            total_false = sum(1 for _, v in votos if not v)

            if total_true > total_false:  # respuesta válida
                validas[jugador] = respuesta
            else:  # respuesta inválida
                puntajes[jugador] += 0
            
            true > false --> respuesta válida
            false > true --> respuesta inválida
            true = false --> ???

        # Paso 2: ver repeticiones entre las válidas
        conteo_respuestas = {}
        for respuesta in validas.values():
            conteo_respuestas[respuesta] = conteo_respuestas.get(respuesta, 0) + 1

        for jugador, respuesta in validas.items():
            if conteo_respuestas[respuesta] == 1:
                puntajes[jugador] += 10
            else:
                puntajes[jugador] += 5

    return puntajes
