- Permite al cliente conectarse y enviar palabras al servidor, que las almacena en una lista. Según una instrucción el servidor envía todas las palabras almacenadas hasta el momento a todos los clientes conectados.
- La comunicación entre cliente y servidor es a través de sockets. 

Cliente:
 - Tiene un socket TCP. Se crean 2 hilos:
    1. uno para 'escuchar' el server, se bloquea.
    2. uno para enviar palabras al servidor

Servidor:
 - Maneja un hilo por cliente
